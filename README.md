# Spock Rock Game

A browser-based **Rock Paper Scissors Lizard Spock** game built with vanilla JavaScript.

## Features

- Play against the computer
- Five choices: Rock, Paper, Scissors, Lizard, Spock
- Live score tracking for player and computer
- Win/lose/tie result feedback
- Confetti effect on player wins
- One-click reset for scores and current round

## Project Structure

- `index.html` – Game markup
- `style.css` – UI styling
- `script.js` – Game logic
- `confetti.js` – Confetti animation helpers

## Run Locally

No build tools are required.

1. Clone the repository:

   ```bash
   git clone https://github.com/LuigiLuigi1/Spock-Rock_Game.git
   ```

2. Open the project folder.
3. Open `index.html` in your browser.

## How to Play

- Click one of the five hand icons to make your move.
- The computer selects a move randomly.
- Scores update automatically after each round.
- Click the reset icon to clear scores and selections.

## Rules

- Rock crushes Scissors and Lizard
- Paper covers Rock and disproves Spock
- Scissors cuts Paper and decapitates Lizard
- Lizard eats Paper and poisons Spock
- Spock smashes Scissors and vaporizes Rock
